FEATURES

  • background/foreground removal
  • generate images with prompts
  • inpaint part of an image with a mask and a prompt
  • segment an image into a mask by clicking on objects
  • edit the AI generated masks, make your own, add transparency or color layers
  • desktop application
  • Background/foreground removal is built-in, no subscriptions or internet required
  • Segmentation using one or more points requires a local server (have not seen APIs that offer this)
  • Inpainting and image generation can be done with a local server or by using your stability AI api key

    • The foreground/background can be removed or they can be auto-filled with a mask, making it fast to edit with inpainting.

    Depending on the parameters, inpainting can benefit from manually extending or removing parts of the generated mask, which can be done with the adjustable brush.
